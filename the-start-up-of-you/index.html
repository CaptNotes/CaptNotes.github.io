《至关重要的关系》摘要与心得

注：【】部分为笔者心得，非原文摘抄。

  • 创造是人的天性，是企业家精神的精髓所在。
  • 现如今的职业忠诚体现在“横向”效忠于职业，而不是“纵向”效忠于老板。
  • 像创业者那样去思考、行动，把你的职业看成自己经营的一项事业。
  • 【企业家需要同时具备优秀的想象力和执行力。】
  • 明知并果敢地承担风险，方可做成大事。
  • 伟大的公司总是处于不断改进和变化之中的。
  • 企业家若要生存，最基本的一点是要比竞争对手占有优势。
  • 产品必须是原创的、绝版的、更快更好或更便宜的，才能占有一席之地。——南希·卢布林
  • 如果你想走一条有别于求职市场上其他求职者的道路，第一步就是要能够把下面的句子补充完整：“一家公司聘用我而不聘用其他人，是以为……”
  • 竞争优势是一切职业策略之根本。
  • 你的竞争优势是由三种各不相同且不断变化的力量相互作用而形成的：你的资产、你的追求/价值观和市场现实，即你的综合能力在竞争市场的供求状况。
  • 工作时在专业项目中能否表现得出类拔萃，和你存折里有多少存款关系不大，关键在于技能、人脉和经历。
  • 热爱自己工作的人比那些只为赚钱而工作的人做得更好更长久。
  • 个性不是你找到的，而是自己浮现的。
  • 无论你的价值观和追求是什么，一定要清楚随着时间的推移它们也在不断发展。
  • “市场”并不是抽象得，而是由人组成的。这些人作出的决定会影响你，而且你必须满足他们的需求。
  • 所有的商业成功都取决于愿意买单的消费者。同样，对于创业启动阶段而言，一切职业成功都取决于你的雇主、客户以及同盟是不是愿意购买你的时间。
  • 如果市场不存在，再聪明也没用。——马克·安德森
  • 不管市场需求是什么，除非你的兴趣和优势能得到发挥，否则你还是无法成为最有竞争力的人。
  • 当你试图平衡雇用市场现实和自身偏好时，妥协在所难免。
  • 要想提高你的竞争优势，最明显的方法就是使你的资产组合多样化并得到巩固。
  • 不要简单地说“我擅长演讲”，你可以这样写：“相比大多数工程师的演讲水平来说，我擅长工程主体的演讲。”
  • 拥有远大的理想抱负至关重要。
  • 想要确定自己是否擅长或者喜欢做某事，需要坚持一段时间才能见分晓。
  • 传统的职业生涯规划只能在相对稳定的条件之下才能奏效。
  • 优秀的企业家会制定计划提高自身的市场竞争优势，但在适当的时候也会灵活地偏离原来的计划，不断朝着真正具有市场竞争优势的方向发展。
  • 【成功无定律。】
  • 真正的计划意味着设计好其中每一个具体步骤。
  • 【知识具有相通性，始终保持兴趣的多元化，有助于更容易获得更多的灵感。】
  • 只有在实践过程中才能找到自己努力的方向以及方法。
  • 热衷于收入高或地位高的工作，也许这些能给我们带来片刻的满足，但却并不能开创有意义的事业。
  • 只需要采取一步行动就能实现的目标也许没有太大意义，或者算不上什么雄心壮志。
  • 对于有相关工作经验要求的工作而言，我们怎么才能获得第一次工作经验呢？解决办法就是：免费做这项兼职工作。
  • 一般而言，最好立足于一次巨大变革的前沿而非后方，这是科技行业的经验之谈。
  • 何时会出现一个转折点来打乱我们的事业，我们无法预料。唯一确定的就是这些会比我们想象中来得更快更突然。
  • 通过建立关系网来帮助自己把握世界。
  • 很少有新创建的公司是靠一个人单枪匹马建立起来的。商界人士都认为组建有才华的团队至关重要。
  • 你的团队是什么样，你的公司就会是什么样。——维诺德·科斯拉
  • 一个稍许平庸但能很好地处理人际关系并为团队做贡献的人比一个百分之百能力但无法进行团队合作的人对公司更有利。
  • 人的行为和信念是相互感染的。
  • 有些人已经拥有你想要的特质，最快改变自己的方式就是和他们在一起。
  • 商界团队的表现往往取决于该团队中最差成员的水平。
  • 盲目醉心于打造关系网，很可能会尝到“职业社交宿醉”的后果。
  • 与人建立真正的社交关系取决于（至少）两个条件：
    • 能从对方的角度看问题；
    • 思考自己如何帮助对方或者与之合作，而不是想着自己能从对方身上得到什么。
  • 朋友并不是你拥有的财产，而是共同维持的关系。
  • 认识新人的最好办法是通过那些你已经认识的人。
  • 很多人在同一时间一般能维持 8-10 个职场同伴。
  • 同盟者的主要特点：
    • 同盟是你经常向其征询意见的人，你相信他的判断能力；
    • 遇到机会，你首先会想到与其分享并合作；
    • 你会把自己的同盟介绍给其他朋友，帮他树立声誉。遇到困难时，你的同盟也会为你做同样的事。
  • 世界上不存在只能同甘不能共苦的同盟关系。如果你们之间的关系没法通过苦难时期的考验，那么你们并不算是同盟。
  • 【无论互联网科技发达到什么程度，面对面交流的优势仍然不可替代。】
  • 很多研究表明，住得不远是两人发展过硬关系的重要因素之一。
  • 当你与合作伙伴慢慢不再清楚记得彼此之间的得失，为对方付出之后不求马上得到回报的时候，你们的关系就由交换关系发展成了真正的同盟关系。
  • 邓巴数字：根据人类大脑皮层的大小，罗宾·邓巴计算出人类一次最多只能跟 150 个人建立关系。
  • 只因为你认识这个人并不意味着他必须得将你介绍给他的另一个朋友。但是，你必须得开口，而且要直接而明确地开口，你需要一个令人信服的理由说服他帮你介绍是有意义的。
  • 考虑你能为你想联系的这个人做些什么，或者至少找出你们之间最紧要的共同利益点。
  • 【交际应该多元化，而不能只想着跟自己相似的人打交道。】
  • 不管你的人生重点是什么，都需要努力建设你的人脉，因为你的职业生涯取决于你能否聪明且大方地对待你所关心的人。
  • 帮助他人意味着你有能力助人。
  • 最好的职场人脉是能为同盟提供真正帮助的人。
  • 不贵但是很体贴的礼物是最好的。
  • 最棒的小礼物的秘诀就在于它是只有你才能提供的东西。
  • 一顿饭比几十封邮件更有用。
  • 如果你想与繁忙的大人物保持联系，需要对地位关系特别注意。地位是指某个时期某个人在某个社会领域里的权利、威望和级别。生活中没有一层不变的权势等级，地位是相对而且呈动态变化的。
  • 迟到是一种典型的权利举动，因为迟到说明“我的时间比你的更宝贵，因此你等我是应该的”。
  • 不要去奉承地位比你高的人，不尊重地位低的人或对权势溜须拍马也要不得。
  • 如果你想和地位比你高的人建立关系，就需要明白对方希望你是个有用的人。
  • 成功都是由机遇造就的。
  • 其他人只是去抱怨，而企业家会问“为什么”。
  • 让关系网成为独一无二、机遇资源丰富的关系网：
    • 每个人都很优秀；
    • 每个群体都有一些共同特征；
    • 地理密度也很重要；
    • 分享和合作风气浓厚。
  • 唯一比加入团队更好的事情就是创办自己的公司。
  • 不要过分急躁，但要快速行动。
  • 风险是动态的。
  • 你永远不会百分之百地有把握。不要将不确定与风险混为一谈。
  • 在你需要某一信息时，及时获取该信息的能力才是真正能让你有所成就的关键。
  • 你的关系网是必不可少的信息来源之一：
    • 首先，人们会提供一些个人言论和印象；
    • 其次，人们可以为你提供个性化的、有针对性的建议；
    • 再次，人们可以为你过滤你从别处收集到的信息；
    • 最后，许多人在与他人交谈时就能够想到更好的想法。
  • 一些如何提出好问题的技巧：
    • 通过交谈而非质问的方式进行提问；
    • 调整问题涉及的范围；
    • 构思与准备问题；
    • 追踪与探索问题。
  • 进行自我投资，对社交网络投资，以及对社会投资。在这三个方面都进行投资，你才有可能最大程度地提升职业潜力。同样重要的是，你才有更大机会来改变世界。
